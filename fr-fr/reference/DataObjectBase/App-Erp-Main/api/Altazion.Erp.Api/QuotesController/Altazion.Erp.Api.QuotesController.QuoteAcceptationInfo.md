## QuoteAcceptationInfo

La classe QuoteAcceptationInfo représente les informations nécessaires pour l'acceptation d'un devis dans l'API des devis clients.

Propriétés publiques :
- Name : Chaîne de caractères, représente le nom associé à l'acceptation du devis.
- Number : Chaîne de caractères, représente un numéro lié à l'acceptation du devis.
