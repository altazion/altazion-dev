Obtenir un bon de prépa

Obtient le détail d'un bon de préparation

Url :[GET] api/logistique/preparation/{bonprepaNum}

Paramètres :

  • bonprepaNum (string)

Url :[GET] api/logistique/preparation/{bprGuid:guid}

Paramètres :

  • bprGuid (System.Guid)

Type de retour : BonPreparationComplet

Type(s) de données :
