Ajouter une url secondaire

Ajoute une url secondaire pour un site

Url :[PUT] api/ecommerce/sites/{idSite}/urls

Paramètres :

  • idSite (int)
  • en tant que body, un objet SiteUrlChange

Type de retour : SiteWebData

Type(s) de données :
