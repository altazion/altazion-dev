## StoreServiceType

The StoreServiceType class represents a type of service in a store.

Public properties:
- Code: a string that gets or sets the unique code of the service type.
- Label: a string that gets or sets the label (name) of the service type.

### TypeScript class
```typescript
interface StoreServiceType {
  /** Gets or sets the unique code of the service type. */
  Code: string;

  /** Gets or sets the label (name) of the service type. */
  Label: string;
}
```
