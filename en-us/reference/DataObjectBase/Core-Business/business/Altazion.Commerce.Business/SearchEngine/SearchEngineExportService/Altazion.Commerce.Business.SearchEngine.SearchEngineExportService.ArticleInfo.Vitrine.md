## Vitrine

The `Vitrine` class represents a showcase linked to an article within the search engine export service.

Public properties:
- `vit_guid` (Guid): Unique identifier of the showcase, in GUID format. This property is used to uniquely identify each showcase.

### TypeScript class
```typescript
interface Vitrine {
  vit_guid: string; // GUID string representing the unique identifier of the showcase
}
```
